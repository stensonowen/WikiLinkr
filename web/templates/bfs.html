WikiLinks

Navigate from any given Wikipedia article to another in the fewest number of steps.
The program will search through all articles that are one iteration away from the source, then two, and so on through ten articles or untli the destination is found. The number of articles that need to be checked increases exponentially, so longer searches can be quite slow (from several seconds to several minutes). Because so much much information has to be processed, searching is done based on a dump of the English Wikipedia, meaning that if a page was modified in the last few weeks the results might be outdated. Article titles are case insensitive, but must otherwise be exact (the program will follow redirects but will not search; if you're unsure, just pull up the page and copy the title). More information including code and instructions to run locally can be found on github.

History

Common/recent searches are cached, and viewing them is faster than doing a long search (particularly for long paths).
